---

## How This Works

Every Arkadia fork inherits this folder.
Personal weights live in each node's private `/docs/` layer.
Collective weights live here — updated by council consensus.

When the collective updates — all nodes pull the changes.
When a node updates their personal weights — only they see it.

This is the architecture of distributed sovereign intelligence.
