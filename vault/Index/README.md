# Arkadia Knowledge Vault — Index

This directory contains auto-generated index files for the vault.
Do not edit manually — generated by the Knowledge OS pipeline.

## Structure

```
vault/
├── Projects/       # Project-scoped notes, conversations, decisions
├── Books/          # Book notes, summaries, highlights
├── People/         # Person profiles, relationship notes
├── Ideas/          # Freeform ideas, concepts, hypotheses
├── Daily/          # Daily notes (YYYY-MM-DD)
├── Research/       # Deep research, citations, analysis
├── Attachments/    # Binary files referenced by notes
├── Templates/      # Note templates (this is a template)
├── Archive/        # Archived notes (soft-deleted)
└── Index/          # Auto-generated search indexes (this directory)
```

## Every note is:
- A Markdown file with YAML frontmatter (human-readable)
- An SQLite record in knowledge/arkadia.db (machine-indexed)
- A graph node in the knowledge graph
- An entry in the immutable timeline
- Chunked and embedded for semantic search

## Never store:
- Orphan notes (every note must have a type and be indexed)
- Secrets, credentials, or tokens
- Binary content inline (use Attachments/)
